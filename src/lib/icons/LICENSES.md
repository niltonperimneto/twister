# Bundled icon licenses

The themed symbolic icon sets in this directory were fetched from the
official upstream icon themes and normalized (hardcoded fills replaced
with `currentColor`, stylesheets/metadata/background plates stripped,
markup minified). Twister itself is GPL-3.0-or-later; all three
licenses below are compatible with bundling under it.

| File | Upstream | License |
| --- | --- | --- |
| `breeze.ts` | [KDE Breeze icons](https://github.com/KDE/breeze-icons) | LGPL-3.0-or-later |
| `adwaita.ts` | [GNOME adwaita-icon-theme](https://github.com/GNOME/adwaita-icon-theme) | LGPL-3.0-or-later or CC-BY-SA-3.0 |
| `cosmic.ts` | [System76 cosmic-icons](https://github.com/pop-os/cosmic-icons) | CC-BY-SA-4.0 |

`default.ts` contains Lucide/Feather-style glyphs drawn for this project
(ISC-compatible originals), used as the universal fallback set.

Attribution: KDE and the Breeze icon authors; the GNOME Project design
team; System76 for the COSMIC icon set.
