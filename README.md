# Twister

Twister is a modern graphical interface for configuring gaming mice on Linux. 
It communicates with `ratbagd` or `ratbagd-rs` — the system daemon that talks to your hardware.

## Development

```bash
# Install dependencies
npm install

# Run the Tauri development app
npm run dev
```

## Build

```bash
npm run build
```
